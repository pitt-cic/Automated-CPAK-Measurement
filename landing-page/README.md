# CPAK Landing Page

Landing page for the Automated CPAK Measurement project - an AI-powered tool for knee alignment analysis.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4
- Motion (Framer Motion)
- TypeScript

## Design

The landing page uses a clinical precision aesthetic with:
- Teal/cyan accent colors evoking surgical precision
- DM Sans + Instrument Serif font pairing
- Light theme with subtle grid patterns
- Animated measurement visualizations
